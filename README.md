#### 项目背景


在web项目开发中往往需要准备403、404、500等报错页面，网络上资源良莠不齐，而且大多数都是收费的，甚至一个页面一收费的那种，前前后后不管是自己开发的还是付费的收集了不少，于是打算整理下，免费分享给大家，以免大家再走冤枉路。如果大家有珍藏的好模板，欢迎大家继续补充贡献！！！


#### 目录结构
